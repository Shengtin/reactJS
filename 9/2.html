User Profile Page

Now that you get your routes are working, let's work on User Profile Page. This page needs to display 2 things:

  1. Username
  2. All images that belong to this user

The route for User Profile page is /user/:id.

When someone visits /users/:id, your app needs to retrieve the value of :id, and make an API call to https://insta.nextacademy.com/api/v1/images?userId=<:id> to get the images.

Challenge

1. Get :id Value

Using React Router documentation, figure out how to use URL Parameter to obtain the :id from URL bar.

Hint: Take a look at the useParams() hook provided by React Router.

2. Making API Call

Once you have the ID, in the UserProfilePage component, you will make API call to:

https://insta.nextacademy.com/api/v1/images?userId=<:id>

and displays all the images belong to that user.

You will need to replace <:id> in the URL with the variable that gives you the user id

3. Display user name

Only the user ID and images are not sufficient. Find a way to display the username as well!
