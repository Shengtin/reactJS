Display Conversations

Create a component where we display all the chat messages. We will use conversations state with this structure:

[
  {username: 'Edwind', message: 'What did the ocean say to another ocean?', timestamp: 1544532325758},
  {username: 'Liren', message: 'sea you later?', timestamp: 1544532341078},
  {username: 'Edwind', message: 'Nothing. It just waved', timestamp: 1544532347412},
  {username: 'Josh', message: "I'm leaving this chatroom", timestamp: 1544532402998},
]

As you can see, conversations should be an array of objects with username, message and timestamp keys.

Timestamp

The value is the result of calling Date.now(), which returns the number of milliseconds elapsed since January 1, 1970 00:00:00 UTC.

When a user sends a message, we call Date.now() and store the value.

To display it in a human readable format, we can use momentjs to format it.

Challenges

1. Display conversations

Render each message nicely along with its username. Remember to handle the case when conversations is an empty array.

2. sendMessage should populate conversations state

When sendMessage is called, get username and messages and push them into conversations array.

3. Scroll down to latest message

Notice how the conversation window won't display the latest chat messages once it overflows? Fix that!

4. User Avatar

Make your chat more lively by generating a profile image for each user. Adorable Avatar API is really fun and easy to use:

Simply call https://api.adorable.io/avatars/150/${username}.png where every username will give you a different image.

The value 150 represents 150px x 150px resolution.
