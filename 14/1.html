Let's Build a Live Chatroom!

Why chatroom?

Without thinking deeply about it, a live chatroom seems like a really simple thing with a form to submit message, and a container to display conversations.

But a chat actually contains lots of frontend logic such as

  • clearing the input after sending a message
  • keep conversation container scrolled to the bottom to always show latest message
  • what happens when your input message is too long
  • what to do when someone joins the chatroom
  • and so on...

And as you already know, React is great at handling complicated UX logic. It's time to put what you have learnt into practice!

Let's start with the Look

Use create-react-app to generate the boilerplate codes:

npx create-react-app next-chat-app
cd next-chat-app
npm start

Sketch out your own design of a group chat room. The design needs to have these components:

1. An area to display all the conversations sent by everyone

2. An area that lists out all the online users

3. A form that allows user to type message and press send

Note: For this chat app, username is automatically generated for each user
