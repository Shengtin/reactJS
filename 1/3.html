Setup: Mac

Go back to previous lesson if you are on Windows

Step 1: Install Homebrew using Terminal

Open up your Terminal app and execute this line:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Homebrew allows you to install other packages/applications through Terminal command line.

Step 2: Install NodeJS using Homebrew

Once the installation is complete, restart your Terminal, then execute the following line:

brew install node

The above command will install NodeJS properly into your computer.

Step 3: Restart Terminal and Check

Close and Reopen Terminal, then use the following commands to make sure you have installed everything successfully:

node -v

It should tell you the version of NodeJS you have installed, as long as it is 8.10.0 and above, then you are good!

Then run this:

npm -v

Node Package Manager comes with NodeJS. Make sure it is at least version 5.2 and above.
