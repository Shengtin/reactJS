Setting Up

Now that you have learnt the basics of Javascript, it's time to jump into modern frontend development - this time, to learn new JavaScript syntax, concepts and terminologies. After which, we will venture into the popular front-end framework React.

Learning React JS would not take long if you have a good grasp of JavaScript skill. Web applications built with this awesome JS framework are specially valued and that also makes web developers with React JS skill high in demand.

What you'll learn

JS in ES6

  • ES6 (ECMAScript 6) became a Javascript standard in June 2015 - it introduces new syntaxes for writing JS; it is crucial for developers to learn ES6 as it is used extensively in many JavaScript libraries, namely React.
  • Estimated time: 0.5 ~ 1 days

React

  • React is popular front-end JavaScript framework for building user interfaces. It allows developers to easily create interactive UIs which update the page on the fly as it detects any data change.
  • Estimated time: 8 days

Setup Your Dev Environment

In order to begin developing JavaScript apps, you'll need to have Node.js - a back-end JavaScript runtime (which allows you to write JavaScript code for servers). In this course however, we will use it mainly in the upcoming React lessons.

Install React Developer Tools (Chrome Extension)

Install React Developer Tool Chrome Extension.

After the installation, open your Chrome Developer Tool and you should find a new 'React' tab.

Anytime you're browsing a React app, this tab will show you tons of useful information about the app.
