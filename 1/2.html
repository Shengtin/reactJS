Setup: Windows

Skip to the next lesson if you are on Mac or Linux

Note: If your user account has a space in the name, ie 'John Smith', we strongly advise you to create a new user account without spaces in the name. eg 'John' or 'JohnSmith'. Some of the software you will be using throughout this course has known issues with with user accounts that have a space in the name. Unfortunately this is out of our hands and there is currently no solution at this time.

Step 1: Install Scoop using PowerShell

For Windows, we will use PowerShell to execute all the commands.

PowerShell comes pre-installed in WIndows 7 and above, if you can't find it, refer to this doc

Open up your PowerShell and execute the following command to install Scoop.

iex (new-object net.webclient).downloadstring('https://get.scoop.sh')

Scoop is a Command Line Installer. It allows us to install other programs through PowerShell commands.

Step 2: Install NodeJS using Scoop

Once the installation is complete, restart your PowerShell, then execute the following line:

scoop install nodejs

The above command will install NodeJS properly into your computer using Scoop

Step 3: Restart PowerShell and Check

Close and Reopen PowerShell, then use the following commands to make sure you have installed everything successfully:

node -v

It should tell you the version of NodeJS you have installed, as long as it is 8.10.0 and above, then you are good!

Then run this:

npm -v

Node Package Manager comes with NodeJS. Make sure it is at least version 5.2 and above.
