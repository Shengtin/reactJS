Building Home Page

You are going to start with the Home Page of your Nextagram.
The way our homepage work is such that it will simply display all the registered users, along with their posts of images.

Something like this but you can come out with your own beautiful designs!

We will hardcode some user data first to display them:

function App() {

  const [users, setUsers] = useState([
    {
      id: 1,
      username: "blake",
      profileImage:
        "http://next-curriculum-instagram.s3.amazonaws.com/idol2-blake.jpg"
    },
    {
      id: 2,
      username: "ryanG",
      profileImage:
        "http://next-curriculum-instagram.s3.amazonaws.com/idol1-ryan.jpg"
    },
    {
      id: 3,
      username: "bigfan",
      profileImage:
        "http://next-curriculum-instagram.s3.amazonaws.com/bigfan-9AE7468E-4C35-41D1-AA68-31939891B5E1.png"
    }
  ]);

  return (
    <div>
      <h1>Home Page</h1>
      <ul>
        {users.map(user => (
          <li>
            {user.id}: {user.username}
          </li>
        ))}
      </ul>
    </div>
  );
}

What's happening in the return() statement?

users is an array. Using .map, we loop through it, and transform each element (an object) into a <li> element.

Inspect the elements on your browser to see the result.

Note that return keyword was omitted from the arrow function because we did not use curly braces {}.


What is happening with the useState?

You should be familiar with how useState works at this point, but just as a recap. We are initializing the users state by passing in an array of objects that contain data about the user. Since we are passing in this array the users variable contain these objects with data about the user. If we want to add or remove users from the array we can use the setUsers function to reassign the users variable. But remember, NEVER MUTATE THE STATE DIRECTLY! Make sure you are making a copy of the array before reassigning it.
