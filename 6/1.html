Nextagram as SPA

React was invented to handle complicated interactions with users within a single page.

Think about the things you can do on Facebook newsfeed page: infinite scrolling, liking and sharing posts, commenting, chat with friends, etc. Technically, the Facebook newsfeed page itself is basically an app, a Single Page App.

Single Page Apps are different from the more traditional multi-page apps that you see everywhere. The biggest difference is that navigating a SPA doesn't involve going to an entirely new page. Instead, your pages load inline within the same page itself:

Building a Instagram Clone (Nextagram) as SPA

We are going to learn most of the React fundamentals by an app called Nextagram.
It's a very basic version of Instagram where you get to see pictures uploaded by all users.

Let's get started by generating a new react app using create-react-app.

You may run npx create-react-app react-nextagram.

Or if you have installed creact-react-app into your machine through npm i -g create-react-app, then you may run create-react-app react-nextagram.
