Our home page is only half done, we fetch users when the app is loaded, now we need to fetch images for each user.

We are going to harness the power of reusability React component.

We will build a <UserImages> component that will be used by each user to display the corresponding images. However, how can we make <UserImages> fetch different images according to users?

According to Nextagram API docs, the endpoint to retrieves images for, say, Blake, is /api/v1/images?userId=3, because Blake's user ID is 3.

Since we are able to get the IDs for all users, we simply need to make <UserImages> receive the user ID as props, and fetch images accordingly.

Challenge

1. Create a UserImages component:

  • Once this component is loaded, you want to make GET requests to retrieve all images for a particular user
  • The API endpoint for this is /api/v1/images?userId=<int>

2. Put this component in HomePage component

  • Pass down userId as prop into UserImages

3. Display loading indicator while fetching images for each user
