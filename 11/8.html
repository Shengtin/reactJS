My Profile Page

With a working log-in flow, complete the last page of your React Nextagram - MyProfilePage.

Start off by creating a new component <MyProfilePage />, then create a routing for it:

<Route exact path="/profile" component={MyProfilePage} />

When this page is loaded, we will call this endpoint:

https://insta.nextacademy.com/api/v1/images/me

to retrieve all the images for current logged in user.

However this API request requires a authorization header with JWT token.

Axios request with header object

Refer to Axios's available config options for making request, attach:

{
  Authorization: `Bearer ${jwt}`
}

to your request header.

Of course, you will first retrieve the JWT from web storage before making the request.

Challenge

Complete MyProfilePage with appropriate loading indicator and error handling. (ie. user visits the page without JWT)
