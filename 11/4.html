It's all great being able to log in and out, but if the user doesn't see any visual cues that they have logged in or logged out, it can be a little confusing and is not great for the user experience.

Thankfully in React, things have been made super easy and there is a great library out there that makes it foolproof to implement notifications and alert messages.

In comes React Toastify

The React Toastify library is made to be as simple as possible to use. Implementation takes no more than 5 lines of code on its most basic implementation, however, it does allow for customizability if you wish to tweek it and make it more suited to your taste.

Before you start, you are going to need to install the package dependencies into your React project.

Go ahead and run the following command:

npm install --save react-toastify

Implementing the Toast notification

If you have reached here, you are either curious as to what lies below, or, something didn't go right during the Toast implementation. Not to worry, let's just take a quick look at how to use the library!

You will be aiming for this outcome by the end!

In the most basic implementation, you need to place the ToastContainer component into your App.js file so that all child components will be able to access the container.

import { ToastContainer } from 'react-toastify';

<ToastContainer />

When you have your ToastContainer component in your App.js, you can now call upon the toast functions to initiate a toast.

Before you do that, there is one more step you need to take and that is to import the styling for the notifications.

Inside of your index.js, add the following line, similar to how you would have done to import the boostrap CSS for your Reactstrap library.

import "react-toastify/dist/ReactToastify.min.css";

Now you can move on to how to use the toast.

Let's use the login as an example.

On successful login you will notify the user that they have logged in successfully via a Toast notification.

axios.post('...', {
... // Your code will be here
})
.then((result) => {
  // Login in with localStorage
  toast.success("Logged in successfully!", {
        position: "top-right",
        autoClose: 5000,
        hideProgressBar: false,
        closeOnClick: true,
        pauseOnHover: true,
        draggable: true
      });
})

You do not need to provide all of the settings, these are optional extras. You can read the documentation to see what settings and options you can change.

The demo website also provides a nice toast builder that let's you try out the different settings and also provides you with the code conversion to implement the notification with those settings.

Go have a play and try for yourself and see what different outcomes your can achieve by just changing a few of the settings and options.

It's great that React has these libraries available to use and there are many more out there to achieve fully functional components right out of the box.
