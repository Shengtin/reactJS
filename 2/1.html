String Interpolation in Javascript

Instead of using + to concatenate strings like this:

let sentence = name + ' drinks ' + count + ' coffee a day.'

You can use Template Literals to make your life easier:

let sentence = `${name} drinks ${count} coffee a day`

Template literals are enclosed by the back-tick (` `) instead of single or double quote. Variables can then be injected using ${variables}.
