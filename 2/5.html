Spread Syntax

The last ES6 syntax, and also the most important syntax that you are going to learn is Spread Syntax (...). This syntax is extremely helpful in React when dealing with data that we want to display.

You use it by writing 3 dots ... in front of a variable.

Spread in destructuring

Array:

const numbers = [1,2,3,4,5]

const [x, y, ...z] = numbers

console.log(x) // 1
console.log(y) // 2
console.log(z) // [3, 4, 5]

As you can see, the ... collects 3,4,5 into an array z.

Object

Same rule applies to object as well, we collect the last 2 properties into rons object.

const fuelPrice = {
  diesel: 2.18,
  ron95: 2.20,
  ron97: 2.47
}

const {diesel, ...rons} = fuelPrice

console.log(diesel) // 2.18
console.log(rons) // { ron95: 2.20, ron97: 2.47 }

Spread to merge

Array:

const x = [1, 2]
const y = [3, 4]

const withoutSpread = [y, x]
console.log(withoutSpread) // [ [3, 4], [1, 2] ]

const withSpread = [...y, ...x]
console.log(withSpread) // [3, 4, 1, 2]

In the above example, you clearly see when we apply ... in front of x or y, they were taken out from their own array.

Here's another example with array of objects:

const onlineUsers = [
  {id: 1, username: 'SweeTzCutie88'},
  {id: 2, username: 'fluffypanda'},
]

const offlineUsers = [
  {id: 3, username: 'XxTHe_FAt4I_FIstErxX'},
  {id: 4, username: 'Datboii69'},
]

// merge both online & offline users:
const allUsers = [...onlineUsers, ...offlineUsers]
console.log(allUsers)
/* prints:
  allUsers: [
    {id: 1, username: 'SweeTzCutie88'},
    {id: 2, username: 'fluffypanda'},
    {id: 3, username: 'XxTHe_FAt4I_FIstErxX'},
    {id: 4, username: 'Datboii69'}
  ]
*/

Object:

In the following example, we merge extraInfo and user object.
However, since ...extraInfo comes after networth, the original networth is replaced with new value:

const extraInfo = {
  gender: 'male',
  age: 12,
  networth: 9000
}

// puts 'extraInfo' into user object
const user = {
  id: 1,
  username: 'SweeTzCutie88',
  networth: 4560,
  ...extraInfo
}

// pay attention to 'networth'
console.log(user)
/* prints
  {
    id: 1,
    username: 'SweeTzCutie88',
    gender: 'male',
    age: 12,
    networth: 9000
  }
*/

Spread to copy

const original = [1,2,3]

const x = [...original]
const y = original

original.push(99)
console.log(x) // [1,2,3]
console.log(y) // [1,2,3,99]

In the code above, y is not a clone of original, it merely points its reference to original, therefore changes made to original will affect y.

However, x is a true clone of original. It doesn't get affected by the changes made on original.

The same rule applies for Object
