Arrow Function

The arrow function is a relatively new introduction to JavaScript in ES6. It allows developers to write more concise functions by removing some unnecessary syntax and refining the usage of this within functions.

Anatomy of the Arrow Function

The arrow function gets rid of unneeded syntax from typical JS function declarations.

// no parameter
const Hello = () => {
  console.log('Hello World')
}

// single parameter
const myFunc = x => x * 2

// with curly braces, you need to return explicitly
const myFuncBlock = (x, y) => {
  return x * y
}

// DOM Event Listener
document.getElementById('header').addEventListener('click', e => {
  e.preventDefault()
  alert('You clicked on header!')
})

Notice how:

  1. The function keyword is no longer needed and is replaced with the =>

  2. The return keyword is needed only when using a code-block

       const multiply = (a, b) => {
     return a * b
   }

    is equivalent to:

       const multiply = (a, b) => a * b

    is also equivalent to:

       const multiply = (a, b) => (
     a * b
   )
