Image Upload Page

What would Nextagram be if you weren't able to take a selfie and share it with the world?

We are going to create our own image uploading form that will talk with the back-end built for you. By the end you should be able to successfully upload an image to the server under the account of the person that is logged in and then be able to display the newly posted image.

Let's go ahead and build our image uploading feature.

Create a new page

Inside of your pages folder in your app. Create a new file called UploadPage.js, this component will house our image uploading logic, and within it we will later create some extra components to help break up our code just a little bit more.

Inside your file create a new class for UploadPage, in here we will handle everything we need to upload images to the server so that they can be publicly viewable and accessible through our other end points.
