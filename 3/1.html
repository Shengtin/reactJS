Learn Command Line

Modern-day Operating Systems such as Windows / Mac / Ubuntu are easy to use. You get beautiful buttons and graphics which makes it intuitive for your average consumer to interact with the computer. There is another way to interact with your computer! Enter the command line / prompt / console / terminal (interchangeable terminologies). We will use the term Terminal since it's shorter.

Because you are no longer an average consumer and are now a young (or young at heart) code padawan, you want full control of your computer through the use of the terminal to get things done!

Challenge

Step 1: Start by opening up your Terminal (Mac) [Use spotlight; cmd + space] or Power Shell (Windows) [Use search]

Step 2: Try the following commands one by one and see what they do then deduce what they each mean

Part 1: Navigating through your folders & files in the terminal

Type the following commands and then hit enter (one-by-one)

Unix Command (MacOS, Linux) PowerShell Notes
pwd pwd
ls ls
cd <folder_or_directory_name> same just pick a folder name based on the names you found from the previous ls command
pwd pwd
cd .. same

Part 2: Manipulating folders & files in the terminal

Unix Command (MacOS, Linux) Power Shell Notes
touch <file_name> New-Item <file_name>
mkdir <folder_name> mkdir <folder_name
rm <file_name> Remove-Item <file_name> WARNING! This command deletes file permanently (does not go through trash / recycle bins)
rm -r <file_name> Remove-Item <file_name> WARNING! This command deletes an entire folder permanently (does not go through trash / recycle bins)

Part 3: Command flags

Most commands on your command line gives you additional options (flags). These flags are usually done by appending <space>-<flag> onto an existing command.

Mac Terminal
Remember ls? Now try ls -la
What do you see? Notice any difference from just using ls?

-la is a flag specific to the ls command

Most commands have flags you can use. To get a full list of flags you can supply with commands, Google <command_name> flags.

For e.g. if you wanted to find all the flags for ls just Google ls flags. Here, click this http://lmgtfy.com/?q=ls+flags

Power Shell
You know ls list out all the files in the current folder, try running the command with flags, ie: ls -Directory and ls -File

For more flags on ls, you can check out this link.

This should get you started! We will learn more commands and flags as we move along.
