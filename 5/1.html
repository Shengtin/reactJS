Handling Events

Handling events with React elements is very similar to handling events on DOM elements.

Without React, you do something like this to handle events such as click:

<button id="btn">You have clicked me <span id="click-count">0</span> times!</button>

<script>
  const myButton = document.getElementById('btn')
  const myClickCount = document.getElementById('click-count')
  let counter = 0

  myButton.addEventListener('click', function(e) {
    e.preventDefault()
    counter++
    myClickCount.innerText = counter
  })
</script>

To do the same thing as above in React, we can create a Clicker component with the following code:

const Clicker = () => {
  const [count, setCount] = useState(0)

  const handleClick = () => {
    setCount(count + 1)
  }

  return (
    <button onClick={handleClick}>You have clicked {count} times!</button>
  )
}

Here are a few notes:

  1. This Clicker component has a state called count.
  2. Each time the button is clicked, the count state is added by 1, and re-trigger the rendering of the button.
  3. onClick is just one of the many events we can put on any React elements, you can check them out here
