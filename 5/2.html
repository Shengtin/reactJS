Styling in React

create-react-app has made it such that you are allowed to import .css files into your JS file. Take a look at the imports of App.js you will see:

import './App.css';

Note that the CSS import was made possible due to the webpack configuration of creat-react-app; it is not a default behavior of all JS files.

There is also another caveat: all the style rules in App.css will be globally applied to every components.

But how about inline-styling? For that, we can use style prop.

Style Property

Every elements in React component can have a style prop that accepts an object:

const btnStyle = {
  backgroundColor: '#0097ab',
  border: 'solid 2px blue'
  // camcelCase property with string value
}

const App = () => {
  return (
    <div>
      <h1 style={{ color: '#09bacd', border: 'solid 2px blue' }}>Hello World!</h1>
      <button style={btnStyle}>Click Me</button>
    </div>
  )
}

Since we are not allowed to write variables with - (dash) in JS, the CSS properties in the style object needs to be in lowerCamelCase, and the values must be in a string.

Other ways to style react components

CSS Modules

create-react-app has a built-in CSS modules feature. It allows you to import CSS to be used for only a particular component (instead of globally applied like regular CSS)

You can learn about it here:
https://facebook.github.io/create-react-app/docs/adding-a-css-modules-stylesheet

Styled Components

styled-components is one of the top picks for React developers.

It allows you to define a component with default styles in it:

import React from 'react'
import styled from 'styled-components'

// we can write actual css syntax
const Button = styled.button`
  background-color: #00973b;
  border: solid 1px black;
`

const App = () => {
  return (
    <Button>Click Me</Button>
  )
}

You can install the styled-components package using the guide below:
https://www.styled-components.com/docs/basics#installation
