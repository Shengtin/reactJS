Calculator: Part 1

Let's get started on creating the calculator. Just stop for a moment and think about a calculator and how you can break it up into smaller pieces.

First, we have our calculator body; this is the top level component for the calculator that holds all the other components that make up the calculator. In the case of our React app, the App.js will be the body, i.e. the container for the rest of the calculator components.

Secondly, in the body we have the Display, the component that shows us the outputs, then finally the Keyboard and the Keys. We can use the actual calculator structure to structure the React app in a similar way.

Create the components

So first of all, you can go ahead and create a display component. What is the display component going to show? It should display the output of the calculation as well as what has been typed on the calculator. We will come back to displaying the output later. For now let's generate the simple structure for the display.

Calculator Display

Create a new folder inside of the src folder in your React project, and name it components. Once you have created a new folder, create a file inside called CalculatorDisplay.js. This is where you will build your display component for the calculator.

We are going to be using a function component to create the display. For now you can just hard code the value into the display, and later on we will make it dynamic.

import React from "react";

const CalculatorDisplay = () => {
  return (
    <div style={{ border: "1px solid black" }}>
      <div style={{ width: "100%" }}>
        <h1 style={{ textAlign: "end" }}>
          0
        </h1>
      </div>
    </div>
  );
};

export default CalculatorDisplay;

What you are seeing shouldn't be too alien; you essentially have a function that returns some HTML content. You will need to get used to writing your code this way as it is the core of React's JSX syntax.

Keypad Component

Again, inside of the components folder, create a new file called KeyPad.js.

import React from "react";

const KeyPad = () => {
  return (
    <div style={{ width: "100%" }}>
      <div style={{ display: "flex", flexDirection: "row", flexWrap: "wrap" }}>
        {/* Later we will put some code in here */}
      </div>
    </div>
  );
};

export default KeyPad;

You may be wondering at this point why we need to import React in the component if we are not even using it. Well we are actually using it as wherever you use the JSX syntax, you need to have React imported into the context of the component.

Quick Recap

  • We have created our first 2 components of the calculator; the display and the keyboard.
  • React components are just JavaScript functions that return HTML content out of them.

Now there are a couple of components to work with, let us move onto the next part. We will see our first case of using hooks in React and also extract some code into a separate file to make it easier to create the keys for the keyboard.
