Let's build a calculator

Let's get stuck in and build a calculator using React.

By the end of the exercise you should have an outcome that resembles something that looks like this:

You will be thrown in the deep end to experience what makes up React. Here are some of the things we will be covering during the exercise:

  • React Hooks
  • Components
  • Passing Props
  • Extracting Code into Seperate Files

Create a new App

Let's go ahead and generate a new app that we can start to build the calculator with.

In your terminal or command prompt, navigate to the directory you want the project to be in and go and run the following command.

npx create-react-app react-calculator

Once the app is created go ahead and navigate into the project folder.

cd react-calculator

Then open the project in your text editor. If you are using VSCode then you can just simply run the following command once you are inside the project folder.

code .

This will open up the entire folder contents inside of VSCode.

Once you have gotten here you can go ahead and start the development server so you can start making changes and writing your code.

Run the following command:

npm start

You should expect the app to open up in the browser automatically when you start the server, if it doesn't you can go to your browser and visit the URL localhost:3000. This is the default server port for React applications.

If you have gotten here, then you are ready to move on to the next part of the exercise.

Let's go!
